功能概述

风控管理用于识别多账号/撞库/刷优惠等异常行为并执行处置。模块位于左侧栏「风控管理」分组，含三个子菜单（各为一个独立页面）：

子功能定位子 Tab核心动作
关联查询按维度查多账号关联，辅助人工研判查会员 / 设备关联 / IP关联查询、导出、表头显示设置（只读分析，无处置）
黑名单对会员/IP/设备做登录、充值、提现等维度的拉黑限制登录 / 充值 / 提现 / IP / 设备 黑名单查询、新增、编辑、删除（二次确认）
刷子监控按关联规则自动识别刷子团伙并批量处罚同关联监控 / 同关联处罚记录查询、默认自动规则、变更规则、批量操作
说明：关联查询是纯只读分析页（不改数据）；处置动作集中在黑名单（人工拉黑）与刷子监控（规则/批量处罚）。三者数据互通：关联查询查到的可疑账号 → 黑名单拉黑 / 刷子监控里查看团伙。

字段控件必填默认取值 / 选项校验规则联动 · 显隐

联动 / 校验 / 状态 汇总

关键联动一览
场景联动规则
黑名单 · Tab 类型会员类（登录/充值/提现）显示财务列与会员ID新增；标识类（IP/设备）显示限制类型列与勾选、筛选多「限制类型」。
黑名单 · 编辑会员类编辑=「修改备注」仅改备注；标识类编辑=改限制类型 + 备注。
登录黑名单 · 差额列仅登录黑名单出现「充/提差额」；充值/提现黑名单无此列。
刷子监控 · 处罚范围全部处罚 → 触发值置灰；只处罚超出范围 → 触发值必填（2-9999）。
刷子监控 · 同关联人数蓝色可点 → 打开同关联会员列表。
关联查询 · 列显隐表头显示设置控制列显隐；导出弹窗独立控制导出字段；主键列不可关闭。
校验规则
  • 黑名单新增：会员类会员ID必填且需校验存在；标识类 IP/设备号必填、限制类型至少勾 1 项。
  • 默认自动规则：处罚范围=只处罚超出范围时，处罚触发值必填且须在 2-9999；=全部处罚时不校验触发值。
  • 破坏性操作：删除黑名单、变更规则、批量处罚均走二次确认弹窗。
